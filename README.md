# Neuronal NB - Deep Learning Notebooks

Una colección de notebooks interactivos para aprender y experimentar con **PyTorch** y **Keras**, dos de los frameworks más populares de Deep Learning.

## Descripción 

Este proyecto contiene notebooks educativos y ejemplos prácticos de redes neuronales implementadas tanto en **PyTorch** como en **Keras/TensorFlow**. Ideal para:

- Aprender conceptos fundamentales de Deep Learning
- Experimentar con arquitecturas neuronales
- Procesar y visualizar datos
- Desarrollar modelos de Machine Learning

## Requisitos Previos

Asegúrate de tener instalado:

- **Python 3.8+**
- **Jupyter Notebook** o **JupyterLab**

## 📦 Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/nickiiinicole/neuronal_nb.git
cd neuronal_nb
```

### 2. Crear un entorno virtual (recomendado)

```bash
python -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate
```

### 3. Instalar dependencias

#### Para PyTorch:
```bash
pip install torch torchvision torchaudio
pip install jupyter numpy matplotlib pandas scikit-learn
```

#### Para Keras/TensorFlow:
```bash
pip install tensorflow keras
pip install jupyter numpy matplotlib pandas scikit-learn
```

#### Instalar todo:
```bash
pip install torch tensorflow keras jupyter numpy matplotlib pandas scikit-learn
```

## 🚀 Uso

### Iniciar Jupyter

```bash
jupyter notebook
```

o para JupyterLab:

```bash
pip install jupyterlab
jupyter lab
```
